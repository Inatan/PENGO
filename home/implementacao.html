Como Foi Desenvolvido


Pengo 3D foi desenvolvido em WebGL na linguagem Javascript, utilizando a biblioteca Three.js, que utiliza o WebGL puro. Também utilizamos a biblioteca DAT.Gui para fazer o menu com as opções de música e com a informação dos controles utilizados no jogo.
Separamos o código em Javascript em diversas classes, uma para os cubos, outra para o pinguim, outra para os patos, etc. Com isso criamos métodos como deletar, criar e mover elementos que se tornaram simples e de fácil manutenção. Adicionamos ao trabalho, com a ajuda da Three.js, explosões com partículas, músicas de fundo, um menu de fácil visualização e sombras.
